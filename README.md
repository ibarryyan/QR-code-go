## 拼图二维码小游戏

Go语言实现的拼图二维码小游戏，在选择图片后进行拼图游戏，拼图成功后生成二维码图片。扫描二维码获得证书。

### 流程图

![./img/main.png](./img/main.png)

### 技术点

- 前端：HTML、CSS、JavaScript
- 后端：Go语言
- [go-qrcode](https://github.com/yeqown/go-qrcode)：二维码生成
- [viper](github.com/spf13/viper)：配置文件读取
